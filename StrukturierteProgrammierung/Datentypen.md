# Skalare Datentypen (speichern einen einzelnen Wert)

## String

*Operanden*
* String name := "Barnabas"
* name := "hello"
* name := name + " world"
* name[0] := 'H'
* name.length (Anzahl der Buchstaben)
* name == name
* name != name
* name < "halt" (wird nach ASCII-Wert verglichen)
* name > "halt" (wird nach ASCII-Wert verglichen)
